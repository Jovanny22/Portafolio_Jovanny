Sistema de Administración para Tienda de Bisutería

Aplicación web para gestionar productos, usuarios y ventas de una tienda de bisutería. Permite subir productos con imagen y precio, activar o desactivar su visibilidad, registrar ventas y administrar cuentas de usuario con diferentes roles.

Tecnologías: C#, ASP.NET Core MVC, Entity Framework, SQL Server

App de Registro de Asistencias con Código QR

Sistema de control de asistencia para empleados mediante lectura de códigos QR. Permite dar de alta y baja a trabajadores, consultar registros por servicio o por empleado y generar reportes en una tabla dinámica.

Tecnologías: C#, ASP.NET Core, Razor Pages, Entity Framework, SQL Server
